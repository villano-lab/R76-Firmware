SSH: Switching and sharing between Windows and Linux

For those already familiar with SSH

I have programmed in the host nicknames 'Mid', 'Middy', and 'Midgardsormr' for the Linux machine. These nicknames also include X-Forwarding as the `-Y` flag. The X-server on this machine is configured to run at startup, so this should work automatically. There is also an SSH key already set up for connecting to Middy. SSH is configured to work through PowerShell, so you should not need to install anything like PuTTY to connect.

For the uninitiated

If you want to run commands on Linux, you will need to either switch computers or follow the steps below.

Using Linux directly

Midgardsormr is also connected to this monitor via HDMI and to the mouse and keyboard using a KVM switch.

On the monitor, press any button (besides the power button) to pull up the main menu, then the second button from the left to open the input selection menu.

The KVM switch is the black box with several cords coming out of it. There is an LED indicator telling you which device you're currently outputting to — these outputs are all labeled.
