Calibration Using CoMPASS

Connect to CAEN HV Unit

Start by launching GECO (pinned to start). It will prompt you for an IP address; this is 134.84.150.53. The username and password are both admin.

In order to enable the channels (turn from red to green), you have to set up the interlock with the module on the backside of the CAEN HV unit. This is a small cylindrical "cap" that goes on the back of the unit.

Once the channels are online, you can swtich one (or many) on. It will turn green and ramp up.

Running CoMPASS

Open CoMPASS and select new or existing project. (On this machine, the calibration is under the 'Documents' folder.) Then, select "Scan," which should automatically connect to the digitizer.

Acquisition Tab

Now that you are in the main menu, set the run ID to NaI-Array-Detector#-#V-Bknd-Datetaken and check that the file format is set to "root".
Checkmark the following settings:

  • raw data (optional - raw data is large so don't mark this unless you need it)
  • save filtered data
  • save unfiltered data
  • save calibrated data
  • save spectra on stop acquisition
  • energy

    • Settings Tab

    In the input tab:

    • Set "Record length" to 5000 (this will round itself to 4992).

    In the discriminator tab:

    • Set "threshold" to 27

    In the synchronization tab:

    • Set "start mode" to software (asynchronous)
    • Set "TRG OUT/GPO Mode" to Level 0

    To collect Data

    1. Select the plot picture at the right end of the upper toolbar.
    2. In the new window, select "new energy histogram" on the left side of the toolbar. This should generate a blank plot.
    3. Click the play button to begin data collection with the detector that is plugged into the desktop digitizer. The plot will periodically update until the run is complete.
    4. When the run is over, you can save an image of the plot using the camera icon or save the data using the floppy disk icon.
